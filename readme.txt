== Description ==

A collection of extra elements for Visual Composer. It was made for Nova premium theme and requires Nova theme installed in order to work properly.

== Installation ==

1. Download the plugin to your desktop.
1. Extract the plugin folder to your desktop.
1. Read through the "readme" file thoroughly to ensure you follow the installation instructions.
1. With your FTP program, upload the Plugin folder to the wp-content/plugins folder in your WordPress directory online.
1. Go to Plugins screen and find the "Nova Addons" in the list.
1. Click Activate to activate it.
